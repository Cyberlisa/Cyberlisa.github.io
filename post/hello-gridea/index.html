Gridea+Github快速教程

2019-08-19 / 1 min read

想知道如何使用Gridea吗？

Git

Git https://git-scm.com

  • 下载后安装
  • 安装完之后在桌面右键Git Bash Here，输入以下内容进行配置
$ git config --global user.name "sailorlisa"
$ git config --global user.email "sailorlisa@outlook.com"

Github

Github账号 https://github.com

  • 每个github账号有1个免费的github pages
  • 如果免费的已被占用（例如hexo），得自行注册新的github账号或者自行买域名，在仓库里的github pages里设置好

仓库

  1. 点击create a new repositories
  2. 输入Repository name：Cyberlisa.github.io。固定格式：.github.io
  3. Description可填可不填。
  4. 勾选Public
  5. 勾选Intialize this repository with a README(可不勾选)
  6. 点击Create repository
  7. 自动跳转到仓库页面，点击Insights后面的Settings
  8. 拉到下面GitHub Pages，看看有没有显示：Your site is published at https://cyberlisa.github.io/
  9. 没有的话稍微等一下再试
  10. 有的话点击链接跳转，看看能否打开

Token

  1. 在github界面点击头像，点击Settings——>Personal settings——>Developer settings
  2. 找到最后的Personal access tokens——>Generate new token——>填写Note——>勾选repo——>Generate token
  3. 创建个txt文档，把token保存起来。如果不保存，那先不要关闭这个页面，待会儿需要用到token。

Gridea配置

下载安装

在https://gridea.dev下载符合自己系统的客户端，安装。

安装完之后打开，如果在【文章】里看到一片空白，卸载整个客户端，重新安装。

填配置

如果打开后能看到默认的2篇文章，点击【配置】：

Platform Github Pages
域名 Github Pages 对应的域名 https://cyberlisa.github.io/
仓库 github上仓库名 Cyberlisa.github.io
分支 Github Pages 对应的分支 master
仓库用户名 仓库所属用户的用户名 Cyberlisa
邮箱 Git 推送时的邮箱 sailorlisa@163.com
Token 上面步骤里的token
CNAME 可不填
  1. 填完配置后，点击【保存】
  2. 无误后，点击【检测远程连接】
  3. 点击【预览】，会弹出浏览器，本地预览效果。
  4. 可在【主题】里选择其他内置主题，不要忘记拉到下面选择【保存】
  5. 选择了其他主题后，点击【预览】
  6. 如果没问题，点击【同步】
  7. 稍等3分钟左右，点击同步按钮下面那个像地球的网格图标，会在浏览器自动弹出博客网页。看看有没有什么问题。
  8. 在客户端里进行任何操作，不要忘记点击【保存】。保存了之后先预览，没问题再同步。
  9. 有问题点击https://github.com/getgridea/gridea里的【Issues】，在【Open】和【Closed】里看看有没有和你问题相似的issue。

Defender

如果你是win10用户，在操作时出现异常，看看通知里有没有defender的警告。我一开始根本操作不了客户端，因为defender限制权限。把defender关了之后才能正常使用。

更换Walker主题

在https://gridea.dev/themes/购买Walker主题，购买完后点击【下载文件】，下载后解压。

点击客户端里的【系统】，找到【源文件夹】，自己打开路径。退出Gridea客户端。

点击【themes】，把walker文件夹复制过来，打开Gridea客户端，打开【主题】，选择主题，滚动下鼠标，选中最后的Walker，保存。
